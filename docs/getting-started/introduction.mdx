## Roles de Usuario

El sistema soporta diferentes roles con diferentes permisos:

### Staff/Recepcionista
- Gestión diaria de citas
- Búsqueda y consulta de pacientes
- Visualización del calendario
- Marcado de citas como completadas

### Admin de Clínica
- Todas las funciones de Staff +
- Configuración completa del sistema
- Gestión de servicios y proveedores
- Configuración de horarios y cierres
- Gestión de suscripciones
- Monitoreo de conversaciones WhatsApp

<Note>
Si es tu primera vez usando el sistema, continúa con [Primer Acceso](/getting-started/first-login) para aprender cómo iniciar sesión.
</Note>
