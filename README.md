# Clinic Management System - Documentation

This repository contains the Mintlify documentation for the Clinic Management System.

## Overview

Documentation for a multi-tenant clinic management system with:
- FastAPI backend
- AI-powered conversational agent (ClinicAgent)
- PostgreSQL database
- WhatsApp integration via Twilio
- Subscription-based billing

## Development

### Prerequisites

- Node.js (for Mintlify CLI)
- npm or yarn

### Local Development

```bash
# Install Mintlify CLI
npm i -g mintlify

# Run development server
mintlify dev
```

The documentation will be available at `http://localhost:3000`.

## Structure

- `docs/` - Main documentation files organized by feature
- `mint.json` - Mintlify configuration
- `_docs-test/` - Test documentation files

## Deployment

This documentation is deployed via Mintlify's platform. See [Mintlify Deployment Guide](https://mintlify.com/docs/deployment) for details.
