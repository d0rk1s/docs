## ¿Qué Son los Horarios de Trabajo?

<Frame>
  <img src="/images/screenshots/work_hours_list_overview.png" alt="Vista general de horarios de trabajo configurados para la clínica y proveedores" />
</Frame>

Los horarios de trabajo definen las **ventanas de disponibilidad** para tu clínica y proveedores. Controlan cuándo se pueden agendar citas en tu sistema.

El sistema de horarios de trabajo admite dos niveles:

1. **Horarios de Clínica**: Definen cuándo está operativa tu clínica
2. **Horarios de Proveedor**: Definen cuándo están disponibles los proveedores individuales (deben intersectar con los horarios de clínica)

<Note>
**Nueva Funcionalidad (2025-11-21)**: Ahora se admiten múltiples períodos no contiguos por día. Puedes configurar turnos divididos como 09:00-13:00 + 15:00-19:00 con pausas para almuerzo.
</Note>

## Jerarquía: Horarios de Clínica → Horarios de Proveedor

El sistema usa una **jerarquía basada en intersección**:

```
Horarios Clínica:    08:00 ━━━━━━━━━━━━━━━━━━━━━━━━━━ 20:00
Proveedor A:                09:00 ━━━━━ 13:00
Proveedor B:                                     15:00 ━━━━━ 19:00
Proveedor C:                09:00 ━━━━━ 13:00 + 15:00 ━━━━━ 19:00

Espacios Disponibles:
  - Proveedor A: 09:00-13:00 ✅
  - Proveedor B: 15:00-19:00 ✅
  - Proveedor C: 09:00-13:00 + 15:00-19:00 ✅
```

<Warning>
Si la clínica está **cerrada** en un día (sin horarios de trabajo configurados a nivel de clínica), NO se pueden agendar citas ese día, independientemente de la disponibilidad del proveedor.
</Warning>

## Numeración de Días de la Semana

El sistema usa identificadores numéricos para los días de la semana:

| Número | Día |
|--------|-----|
| 1 | Lunes (Monday) |
| 2 | Martes (Tuesday) |
| 3 | Miércoles (Wednesday) |
| 4 | Jueves (Thursday) |
| 5 | Viernes (Friday) |
| 6 | Sábado (Saturday) |
| 7 | Domingo (Sunday) |

## Múltiples Períodos por Día

Puedes configurar múltiples períodos de tiempo no contiguos para el mismo día:

**Ejemplo: Turno Dividido**
```
Lunes (1):
  Período 1: 09:00-13:00 (Mañana)
  Período 2: 15:00-19:00 (Tarde)
  Espacio: 13:00-15:00 (Pausa para almuerzo - SIN citas)
```

<Info>
El sistema genera espacios **solo dentro de los períodos configurados**, no en los espacios vacíos. Esto asegura una programación profesional alineada con tu disponibilidad real.
</Info>

## Validación de Solapamiento

El sistema **previene períodos de tiempo solapados** para la misma clínica/proveedor/día de la semana:

- **Fórmula de Solapamiento**: `A_inicio < B_fin AND B_inicio < A_fin`
- **Períodos Contiguos**: Permitidos cuando `A_fin == B_inicio` (ej., 09:00-13:00 y 13:00-17:00)
- **Validación**: Se realiza a nivel de aplicación antes de guardar

**Ejemplos**:

✅ **Válido**: 09:00-13:00 + 15:00-19:00 (espacio 13:00-15:00)

✅ **Válido**: 09:00-13:00 + 13:00-17:00 (contiguos)

❌ **Inválido**: 09:00-14:00 + 13:00-17:00 (solapamiento 13:00-14:00)

## Mejores Prácticas

<AccordionGroup>
  <Accordion title="Comienza con los horarios de clínica primero">
    Siempre configura los horarios a nivel de clínica antes de establecer los horarios de proveedor. Los horarios de proveedor se validan contra los horarios de clínica.
  </Accordion>

  <Accordion title="Usa múltiples períodos para turnos divididos">
    En lugar de un período largo con bloqueo manual, usa múltiples períodos (ej., 09:00-13:00 + 15:00-19:00) para manejo automático de espacios.
  </Accordion>

  <Accordion title="Revisa los errores de validación de solapamiento">
    Si obtienes errores de solapamiento, verifica la fórmula: `A_inicio < B_fin AND B_inicio < A_fin`. Ajusta los tiempos para crear un espacio o hacer períodos contiguos.
  </Accordion>

  <Accordion title="Usa cierres para cambios únicos">
    Para días festivos o días excepcionales, usa la funcionalidad de **Cierres** en lugar de eliminar horarios de trabajo. Los horarios de trabajo definen tu horario semanal regular.
  </Accordion>

  <Accordion title="La invalidación de caché es automática">
    Cuando creas, actualizas o eliminas horarios de trabajo, el sistema automáticamente limpia las cachés de disponibilidad. No se necesita acción manual.
  </Accordion>
</AccordionGroup>

## Cómo Afectan los Horarios de Trabajo a la Disponibilidad

El sistema de disponibilidad sigue esta lógica:

1. **Obtener horarios de trabajo de la clínica** para el día de la semana solicitado
2. **Obtener horarios de trabajo del proveedor** para el mismo día de la semana
3. **Calcular intersección** de los horarios de clínica y proveedor
4. **Generar espacios** dentro de los períodos intersectados (ej., cada 30 minutos)
5. **Filtrar conflictos** (citas existentes, cierres, calendarios externos)

**Ejemplo**:

```
Solicitud: Disponibilidad para Lunes (1), servicio de 30 minutos

Horarios Clínica (Lunes): 08:00-20:00
Horarios Proveedor (Lunes): 09:00-13:00 + 15:00-19:00

Intersección: 09:00-13:00 + 15:00-19:00

Espacios Generados:
  09:00, 09:30, 10:00, 10:30, 11:00, 11:30, 12:00, 12:30
  15:00, 15:30, 16:00, 16:30, 17:00, 17:30, 18:00, 18:30

NO se generan espacios: 08:00-09:00, 13:00-15:00, 19:00-20:00
```
